# AviationDB
AviationDB — это зачетный проект по предмету "Программирование на Python", представляющий собой оконное приложение для управления базой данных авиарейсов. Проект реализован с использованием архитектуры MVC (Model-View-Controller) и предоставляет полный набор CRUD операций для работы с данными.
