# AviationDB
AviationDB — это зачетный проект по предмету "Программирование на Python", представляющий собой desktop-приложение для управления базой данных авиарейсов, разработанное на Python с использованием:
Архитектуры MVC (Model-View-Controller) для четкого разделения ответственности

Полного набора CRUD-операций для работы с данными

Графического интерфейса на Tkinter

Технологии psycopg2 для взаимодействия с базой данных PostgreSQL
